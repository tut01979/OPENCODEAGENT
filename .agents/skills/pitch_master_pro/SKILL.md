# 🎨 OpenCode Pitch Master: La Factoría de Persuasión v1.0

Esta habilidad transforma una idea cruda en un "Pitch Deck" irresistible para socios, inversores o clientes.

## 🧠 Estructura Narrativa Maestra (The Hook Framework)
Para cada presentación, el agente debe estructurar el contenido siguiendo este orden psicológico:

1.  **⚡ El Gancho (The Hook):** Un titular de 10 palabras que defina el valor masivo.
2.  **📌 El Dolor:** Descripción visual y emocional del problema que resolvemos.
3.  **💎 La Solución:** Cómo el producto elimina ese dolor de forma única.
4.  **✨ El "Momento Wow":** Generación de una imagen o mockup premium del producto.
5.  **📈 Potencial de Mercado:** Por qué ahora es el momento perfecto.
6.  **🤝 El Cierre (The Call to Action):** Instrucciones claras de qué queremos que haga el cliente.

## 👁️ Estándar Visual "Nano Banana 2.0"
Al sugerir o generar imágenes, el agente debe aplicar este estilo estético:
*   **Contrastes Elevados:** Fondos oscuros (o blancos puros) con detalles en colores neón/tecnológicos (azul cobalto, esmeralda, ámbar).
*   **Glassmorphism:** Uso de texturas de cristal, transparencias y desenfoques (frosted glass).
*   **Minimalismo Funcional:** Mucho espacio en blanco (`whitespace`), tipografías modernas (Inter, Outfit, Roboto) y ausencia de ruido visual.
*   **Claridad 4K:** Perspectivas cinematográficas, iluminación de estudio y realismo fotográfico en mockups.

## 🛠️ Herramientas Sugeridas
1.  **`generate_image`**: Para crear mockups de alta fidelidad, banners de landing page y escenas de uso.
2.  **`read_file / read_url`**: Para analizar la competencia del proyecto y encontrar ángulos únicos.
3.  **`text_to_speech`**: Para crear audios de "pitch" profesionales si el usuario lo requiere.

## 📜 Reglas de Oro del Presentador
*   **No aburras:** Menos texto, más impacto visual.
*   **Sé específico:** No digas "es el mejor", muestra por qué es el mejor con datos o escenas.
*   **Adáptate:** Si es un producto físico, usa iluminación cálida; si es software/SaaS, usa estética "Cyber-Professional".
